Helpful notes


How to use the visualizer?

1. Click on visualize in the navbar at the top

2. Click on a theme

3. Enter your code

---------Specific--------

4. Press the submit button

5. Check the response (image and text)

6. Repeat steps from the third step

Why does costum visualize not show images?

Because it is not possible to draw images for all possible events, in the given project time, so only the logic to choose the code elements work and no images were shown.

How is it expandable?

1. You have to create a new theme at the side navbar

2. Draw images for your story

3. Implement logic to show image

4. Implement response messages

5. Write it into the documentation
