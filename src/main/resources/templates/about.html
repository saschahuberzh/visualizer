About


In diesem Projekt geht es darum, dass wir unsere Projektidee für die BMA Arbeit genauer spezifizieren. Das heisst, dass wir hier das Konzept für das Projekt ‘Code Visualizer’ erstellen und dabei auch noch genauer auf unsere zentrale Frage ‘Wie kann man Code visuell darstel-len? ’ eingehen.



Idea


Die Idee dieses Projektes ist es, das Programmieren und Zeichnen zu verbinden. Dadurch wollen wir erreichen, dass man Code jemandem ein wenig besser erklären kann, der vielleicht vorher auch noch nicht viel damit zu tun hatte. Der Name Code Visualizer spricht hier eigent-lich für sich. Wir wollen also Code visualisieren, indem wir ihn mit Zeichnungen / Animationen darstellen. Das Ziel wird hierbei sein, dass der Benutzer in einer Applikation entweder direkt Code eingeben kann oder geleitete Aufgaben ausführt und dabei direkt die Visualisierung sieht. Da man hier aber nicht für jeden Fall eine Zeichnung machen kann, werden wir hier eher in Richtung Stories gehen. Das heisst wir geben dem Benutzer bestimmte Geschichten be-reits vor. Durch diese Geschichten können wir dann auch eine bessere Verbindung aus Zeichnung und spezifischen Code Elementen erschaffen und dem Benutzer so gut einzelne wichtige Teile des Programmierens zeigen.
