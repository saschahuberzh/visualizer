About


The aim of this project is to specify our project idea for the BMA work. This means that we are specifying the concept for the "Code Visualizer" project and implement the functions to visualize code.



Idea


The idea of ​​this project is to combine programming and drawing. Through this we want achieve that you can explain code a little better to someone who may not have had a lot to do with it. The name Code Visualizer actually speaks for itself. So we want Visualize code by showing it with drawings / animations. The goal here will be that the user can either enter code directly or guided tasks in an application executes and sees the visualization directly. Since you don't have a drawing here for every case, we will go more in the direction of stories here. That means we give the user certain Stories already prepared. Then through these stories we can also make a better connection drawing and specific code elements create and provide the user with individual important parts of the programming.
